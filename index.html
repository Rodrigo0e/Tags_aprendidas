<p>

(Voltar)

Aprendi que com a tag <p> criamos um paragráfo.

<h1> a <h6>

(Voltar)

As tags <h1> a <h6> em HTML são usadas para definir títulos e subtítulos, organizando o conteúdo hierarquicamente.

<input type="text"/>

(Voltar)

Cria um campo para digitar.

Atributo id=""

(Voltar)

É um identificador, geralmente usado no javascript para alterar uma linha de texto no HTML.

Atributo style

(Voltar)

É usado para adicionar algo do CSS diretamente pelo HTML.

class

(Voltar)

É usada para criar uma classe que é interligada com CSS.

<img src="">

(Voltar)

Adiciona uma imagem na página.

width

(Voltar)

Define a largura de um elemento.

<blockquote>

(Voltar)

Cria uma citação.

<mark>

(Voltar)

Marca o texto.

<sup>

(Voltar)

Faz com que a letra ou texto fique um pouco acima dos outros.

<a>

(Voltar)

Ela é uma ancora usada para criar um link para outra página.

title

(Voltar)

É um atributo, que quando passar o mouse em cima do texto, vai aparecer uma frase.

target

(Voltar)

O taget="_self" abre um link na mesma aba. Já o target="_blank" abre o link em outra aba.

<hr>

(Voltar)

Cria uma linha na página.

<small>

(Voltar)

Cria um texto pequeno.
